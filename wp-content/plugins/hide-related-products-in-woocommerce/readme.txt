== Description ==

Don't want to show Related Products in your WooCommerce store ? Hide them with this plugin.

After Installing this plugin, Related Products from your product page will be removed. You can contact me on <a href="http://basitansari.com/">my website</a> for further support.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `hide-related-products.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You are done.
